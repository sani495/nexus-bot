# Debt

### What is Debt?

Debt is a term used to describe the state of owing money or usually money owed to another . For an example, let's say you have a credit card and you use a lot of money pass your limited amount. You owe money back depending on how much you spent.

### How can I go into debt? How can I fix that?

Nexus can now detect what you did to put your whole server in debt. This is caused by the **"add"** command which counts as cheating. When your server goes into debt, there will be limitations within the **Configuration**&#x20;

{% hint style="info" %}
Every time the command `add` used, a credit is to the debt until it's fixed by staff! :)
{% endhint %}
